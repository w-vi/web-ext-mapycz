    About

    This is a very simple WebExtension extension to find selected text on https://mapy.cz/ . It just opens new a tab with the https://mapy.cz/ search result.

    It is for those who are using mapy.cz and find it annoying to copy&paste the address to the mapy.cz search.

    From version 1.1.0 it also supports navigation/search bar intergration using the @mapy

    Install

    Chrome/Chromium: Get it directly from chrome webstore.

    Firefox 57+: Get it directly from Mozilla Addons.

    Or grab it from source on Github.

    Support

    Please use either the chrome webstore support or github issues
